# Mauritania Explorer

Interactive illustrated SVG map of Mauritania — [mauritania.abbad.app](https://mauritania.abbad.app)

## Features
- Clickable places: Iron Ore Train, Eye of the Sahara, Banc d'Arguin, Chinguetti, Oualata, Atar, Terjit, Tichit, Nouakchott, Ben Amera
- Random music (Dimi Mint Abba + DJ Dahker)
- World location inset
- Mobile responsive
